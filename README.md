# MEPHI Session Project 2026

Репозиторий содержит материалы по сессионному заданию по Linux/UNIX.

## Основной файл для проверки

Главный файл: **`report.txt`**.

В нём последовательно размещены:
1. очищенная история команд, относящихся только к выполнению ДЗ;
2. содержимое `/etc/passwd`;
3. содержимое `/etc/group`;
4. результаты финального тестирования.

## Что реализовано

В рамках задания были выполнены:
- настройка статического IP и hostname;
- установка `nginx`, `tcpdump`, `libcap-ng-utils`;
- создание и монтирование файловой системы `ext4` с меткой `MEPHI_DATA`;
- запуск и автозапуск `nginx`;
- создание пользователя `mephi-admin` и группы `mephi-devs`;
- настройка прав доступа к `/data/mephi-web`;
- настройка SELinux-контекста для web-каталога;
- назначение capabilities для `tcpdump`;
- ограничение входа для `root`;
- публикация и проверка web-страницы.

## Основные артефакты

- `network_check.txt` — проверка сетевой доступности;
- `fstab.txt` — настройка автомонтирования;
- `nginx_recent_logs.txt` — проверка запуска `nginx`;
- `permissions.txt` — владелец, группа, права и контекст каталога `/data/mephi-web`;
- `users_groups.txt` — сведения о пользователе `mephi-admin` и группе `mephi-devs`;
- `selinux_status.txt` — режим SELinux;
- `file_contexts.txt` — SELinux-контекст каталога;
- `tcpdump_capabilities.txt` — назначенные capabilities;
- `index.html` — содержимое web-страницы;
- `curl_output.txt` — результат проверки страницы;
- `mephi-nginx-screenshot.png` — скриншот проверки страницы;
- `tcpdump-4.99.6-2.fc43.aarch64.rpm` — RPM-файл, использованный в задании.

## Примечание

В `report.txt` используется очищенная история команд, относящихся только к выполнению ДЗ. Технические проверки, опечатки и вспомогательные команды исключены для удобства проверки.
